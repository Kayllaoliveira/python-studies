# python-studies

A personal Python learning repository following the "Curso em Vídeo" by Gustavo Guanabara.

## Project Structure

- `main.py` — Entry point / main runner script
- `Curso em vídeo-Gustavo Guanabara/` — Python exercises from the course
  - `Mundo1-001.py` — Basic print/hello world exercise
  - `Mundo1-002.py` — User input exercise

## Language & Runtime

- Python 3.12

## Running

```bash
python3 main.py
```

## Notes

- The exercise files use `//` as comment syntax (invalid Python), which is a beginner mistake in the source files. The `main.py` entry point contains corrected runnable code.
